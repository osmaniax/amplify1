[git手順]

vi index.html
git add index.html
git commit -m "Commit Comment"
git push origin master

[反映時間]

1.カスタムドメインのDNS作成後のドメイン反映は30分程度
元のオリジンドメインは「 https://master.d31c74106lj4gh.amplifyapp.com/」
2.コンテンツ更新後にgit push、デプロイの反映は2分程度
※更新をしばらく行っていない場合、デプロイに時間がかかる。コンテナサーバ起動のためか？
